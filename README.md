# qr-code-generator
Сервис для быстрого и удобного создания красивых QR-кодов. Пользователь вводит текст или URL, настраивает внешний вид (цвет, размер, добавляет логотип) и получает готовый QR-код, который можно скачать или поделиться. Такой инструмент полезен для маркетинга, визиток, меню и просто для повседневного использования.
